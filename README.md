🚗 Automobile Price Prediction System

Overview

The Automobile Price Prediction System is a machine learning project that predicts the market price of vehicles based on various automobile features. The model analyzes attributes such as company, model, manufacturing year, fuel type, transmission type, and kilometers driven to estimate the selling price accurately.

Features

- Predicts automobile prices based on vehicle specifications.
- Performs data preprocessing and feature engineering.
- Uses machine learning algorithms for accurate predictions.
- Provides an interactive and user-friendly interface.
- Helps buyers and sellers determine fair market values.

Installation

1. Clone the repository:

git clone https://github.com/your-username/automobile-price-prediction.git

2. Navigate to the project directory:

cd automobile-price-prediction

3. Install dependencies:

pip install -r requirements.txt

4. Run the application:

streamlit run app.py
